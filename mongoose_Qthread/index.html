RESTFUL JSON - Proof of Concept

by Central Services

This test page is a proof of concept of Restful web services using json to build RESTful APIs

TEST 1:
Push the button and get a nym and Public Key
 
 

This is an example of how the Rest web service can be invoked:

$('.GetNymButton').click(function(){
$.ajax({
url: 'http://localhost:38000/api/getpubkey',
method: 'GET',
dataType: 'json',
data: { role: $('#role').val()},
success: function(json) {
$('#pubkey').html(json.result);
}
});

$.ajax({
url: 'http://localhost:38000/api/getnym',
method: 'POST',
dataType: 'json',
data: { role: $('#role').val()},
success: function(json) {
$('#nymid').html(json.result);
}
});
});
 TEST 2:
Enter numbers below and see the output. Browser will send two numbers to /api/sum URI. Mongoose hit the bussines logic to execute the "multiplication" function and return the sum result

 

This is an example of how the Rest web service can be invoked:

jQuery(function() {
$(document).on('keyup', '#n1, #n2', function() {
$.ajax({ url: 'http://localhost:38000/api/mult',
method: 'POST',
dataType: 'json',
